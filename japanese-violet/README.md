# URLshortener
URLshortener by Web Engineering written in Node.js @Unizar


Tecnologias
1. MongoDB
2. Docker
3. Express
3. Node.js

# Run server
## NPM
```
$> npm start
```
## Docker-compose
You need Linux or OS X

```
$> docker-compose up -d
```

# Run test
## Npm
```
$> npm test
```

##Docker

```
$> docker-compose run server npm test
```
