# URLshortener
URLshortener by Web Engineering written in Node.js @Unizar


Tecnologias
1. MongoDB
2. Docker
3. Express
3. Node.js

# Dependencias
Ubuntu/Debian/Elementary OS

```
sudo apt-get install -y libcairo2-dev libjpeg8-dev libpango1.0-dev libgif-dev build-essential g++
```

OS X

```
brew install cairo
```

# Run server
## NPM
```
$> npm start
```

# Run test
## Npm
```
$> npm test
```
